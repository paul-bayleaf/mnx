The staff labelref object

Type: String

This data encodes the answer to the question: "should we display the part's name or the part's shortName"?

Allowed values:

Value Description
"name" Use the part's "name" attribute.
"shortName" Use the part's "shortName" attribute.

Parent objects

This object is used by the following parent objects:

  • staff: "labelref"
  • staff source: "labelref"
